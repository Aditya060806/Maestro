# Maestro Framework

## Document Conversion Test

This is a **test document** to demonstrate the Pandoc conversion capabilities of the Maestro Framework.

### Features

- Convert various document formats to markdown
- Batch processing capabilities
- AI-optimized output formatting
- Automatic format detection

### Supported Formats

1. Microsoft Word (.docx, .doc)
2. PDF documents
3. HTML files
4. EPUB books
5. LaTeX documents

> The Maestro Framework makes document conversion seamless and efficient for AI assistant processing.

### Code Example

```bash
bash .agents/scripts/pandoc-helper.sh convert document.docx
```

For more information, visit the [Maestro repository](https://github.com/Aditya060806/Maestro).
